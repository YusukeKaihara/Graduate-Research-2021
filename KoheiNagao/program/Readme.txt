make_api_column.py : FFRI Dataset2013-2017から各マルウェアファミリとその検体数を得るプログラム。
		     カレントディレクトリにMWS2020があれば実行可能。
		     出力結果はDataset、その情報に当たるInfoText。


api_graph.py : 作成したデータセット内の検体のAPIコール列の繰り返しを有向グラフで可視化するプログラム。
	       検体はソースコード内で絶対パスで指定しているため、適宜書き換えて下さい。
	       出力結果は指定した検体のグラフ。


random_choice.py : 無作為に検体を選ぶためだけのプログラム。
		　 入力、出力共にソースコード内で指定しているため、適宜書き換えて下さい。
		   


LZW_trie.py : APIコール列を辞書式圧縮アルゴリズムLZWによって圧縮するプログラム。
	      このプログラムではAPIコール列をソースコードのメイン関数内で指定しているためLZWの実装ができたかの確認のために作成した。
	      出力は辞書番号の列。

LZT_trie.py : LZW_trie.pyと同様。LZT_classification.pyに内包されている。


LZT_clasification.py : LZTの辞書を用いて亜種分類の正解率を出力するプログラム。
		       Excelに各マルウェアファミリごとの正解率が出力される。
